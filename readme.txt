This plugin will allow your customers to post feedback about their shopping experience on ShoppingGate for WooCommerce

== Description ==

ShoppingGate is a Danish customer feedback service that is free for everyone to use. The most important thing about ShoppingGate is trust.

Therefore we have created this plugin that confirmes orderids of every feedback posted on your profile at ShoppingGate.

Customerfeedback is important for your business, choose wisely and correctly. Choose ShoppingGate.

Thanks for using ShoppingGate for WooCommerce

Contact:
http://shoppinggate.dk/kontakt

== Installation ==


1. Upload the plugin files to the `/wp-content/plugins/shoppinggate` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Click on the ShoppingGate menu item (left in the administration panel)
4. Fill in the correct information and click save. Check for the API Message to return with a message saying the API information is correct.
